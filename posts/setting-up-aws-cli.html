Setting up AWS CLI

First download and install the cli with pip.

pip install awscli

You'll need a secret key and private key. To get these you'll need to setup a IAM user. Go to https://console.aws.amazon.com/iam/home?#/users.

Click "Add User". Give it a username "cli" and click "Programmatic Access". On the next screen click "Attach Policy" and search for AdministratorAccess. Attach that Policy and click "Create User".

aws configure

Now you'll have a Access key ID and Secret access key. Enter those when prompted.

That's it!
